Diferencias entre bibliotecas, framework y APIs

Biblioteca Framework API
Diferencias Conjunto de funciones, clases o rutinas que están diseñadas para ser utilizadas por otros programas. Estructura o infraestructura que proporciona una base para el desarrollo de aplicaciones. Una API define cómo interactuar con un sistema o componente de software.
Proporciona funcionalidades específicas que pueden ser invocadas por el código del programa principal. Define una arquitectura, un conjunto de patrones de diseño y reglas que guían el desarrollo de aplicaciones. Proporciona un conjunto de reglas y definiciones que permiten a las aplicaciones comunicarse entre sí o con servicios externos.
Las bibliotecas son independientes y no imponen una estructura particular en la aplicación que las utiliza. Los frameworks suelen ser más completos que las bibliotecas y pueden abordar múltiples aspectos de una aplicación, como la gestión del ciclo de vida, la seguridad, el enrutamiento, la persistencia de datos, etc. Las API pueden ser parte de una biblioteca o framework, o pueden existir de manera independiente.
Ejemplo de uso en Java Apache Commons Lang es una biblioteca popular que proporciona utilidades adicionales para trabajar con cadenas, fechas, números y otros tipos de datos en Java. Spring es un framework de aplicación Java que proporciona una infraestructura completa para el desarrollo de aplicaciones empresariales. En este ejemplo, utilizaremos Spring Boot para crear un simple servicio web. JDBC es una API estándar de Java que define cómo interactuar con bases de datos relacionales. En este ejemplo, utilizaremos JDBC para conectarnos a una base de datos MySQL y ejecutar una consulta simple.
Ejemplo de uso en JavaScript jQuery es una biblioteca de JavaScript que simplifica la manipulación del DOM y la interacción con AJAX en las aplicaciones web. React es un framework de JavaScript para construir interfaces de usuario interactivas. Fetch API proporciona una interfaz JavaScript para acceder y manipular partes del canal HTTP, como peticiones y respuestas.
